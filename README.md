# Algorithms

Collection of algorithm challenges solved in TypeScript.

## Problems

| Problem | Technique | Directory |
|---------|-----------|-----------|
| Valid Palindrome | Two pointers on a string, checking characters from both ends | `palindrome/` |
| First Unique Character in a String | Count characters, then find the first one that appears once | `unique-character/` |
| Valid Palindrome II | Like palindrome check, but you may delete at most one character | `palindrome-II/` |
| Contains Duplicate | Classic hash-set problem: determine whether any value appears at least twice | `duplicate-chars/` |
| Ransom Note | Use frequency counting to check whether one string can be built from another | `ransom-note/` |
